# RUF-MPP

RUF-MPP is a graph neural network framework for multi-task molecular property prediction. The framework supports balanced scaffold splitting, rule-based molecular augmentation, multi-task training, Min-AUC-First model selection, TensorBoard logging, and multi-run result aggregation.
